:::note
This page covers how to convert existing code from Java Spec API to Kotlin API. For more information, refer to the [Tutorial](../tutorial/overview.md).
:::

| Java Spec API | Kotlin API |
| -- | -- |
| @LayoutSpec | [KComponent](basics.mdx) |
| @OnCreateLayout | [ComponentScope.render()](basics.mdx) |
| @OnCreateTransition | [useTransition()](/kotlin/hooks-for-spec-developers.mdx#usetransition) |
| @OnCreateInitialState / @OnUpdateState | [useState()](/mainconcepts/use-state.mdx) |
| Lazy State | [useRef()](mainconcepts/use-ref.mdx) |
| @OnError | [useErrorBoundary()](/mainconcepts/use-error-boundary.mdx) |
| @OnAttached / @OnDetached | [useEffect()](/mainconcepts/use-effect.mdx) |
| @OnEvent | [Event Handling](event-handling.mdx) |
| @OnCalculateCachedValue | [useCached()](https://fburl.com/code/s2xoyg3o) |
| @TreeProp | [getTreeProp()](basics.mdx#treeprops) |
| @OnCreateTreeProp | [TreePropProvider](basics.mdx#treeprops) |
| Common Props | [Style](basics.mdx#common-props) |
| @InjectProp | [use new Static DI API](https://www.internalfb.com/intern/wiki/Ultralight/Static_Injector/) |
| Column.create(context) | [Column()](flexbox-containers.mdx); see [Flexbox Cheatsheet](flexbox-containers.mdx#flexbox-properties-cheatsheet) |
| Row.create(context) | [Row()](flexbox-containers.mdx); see [Flexbox Cheatsheet](flexbox-containers.mdx#flexbox-properties-cheatsheet) |
| Manual Keys | [key()](mainconcepts/coordinate-state-actions/keys-and-identity.md#assigning-manual-keys) |
| Handle | [handle()](basics.mdx#handles) |
| @OnCreateLayoutWithSizeSpec | not yet supported: [suggested workaround](../custom-layout.md#kotlin-integration) |
| @MountSpec / @GroupSectionSpec | not yet supported: use Spec API for now |
| @OnTrigger | not yet supported; use Spec API for now |
