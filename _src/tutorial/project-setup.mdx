This page provides a set of steps that help you establish the necessary settings and dependencies required to add Litho to your project.

## 1. JCenter repository

Litho artifacts are published to Bintray's JCenter. So, you'll need to make sure you have the JCenter repository in your root `build.gradle` file:

```groovy title="build.gradle"
repositories {
  jcenter()
}
```

## 2. Add Litho core dependencies

<VersionedCodeBlock language="groovy" code={`
dependencies {
  // Litho
  implementation 'com.facebook.litho:litho-core:{{site.lithoVersion}}'
  implementation 'com.facebook.litho:litho-core-kotlin:{{site.lithoVersion}}'
  implementation 'com.facebook.litho:litho-widget:{{site.lithoVersion}}'
  implementation 'com.facebook.litho:litho-widget-kotlin:{{site.lithoVersion}}'
  kapt 'com.facebook.litho:litho-processor:{{site.lithoVersion}}'
  // SoLoader
  implementation 'com.facebook.soloader:soloader:{{site.soloaderVersion}}'
  // Testing Litho
  testImplementation 'com.facebook.litho:litho-testing:{{site.lithoVersion}}'
}
`} />

In oder to use dependencies with annotation processors, you need to apply the Kotlin KAPT plugin at the top of your application's `build.gradle` file:

```groovy
apply plugin: 'kotlin-kapt'
```

## 3. Wire up native libs

As a last step, you need to initialize `SoLoader`. Litho has a dependency on [SoLoader](https://github.com/facebook/SoLoader) to help load native libraries provided by the underlying layout engine, [Yoga](https://yogalayout.com/docs/).

Your custom `Application` class is a good place to do this:

```kotlin title="MyApplication.kt"
class MyApplication: Application() {

  override fun onCreate() {
    super.onCreate()
    SoLoader.init(this, false)
  }
}
```

## Testing your setup

Having set up the project, you can test the config by adding a simple UI created with Litho. The simple UI is a built-in `Text` widget that displays the "Hello World!" text to an activity:

```kotlin file=sample/src/main/java/com/facebook/samples/litho/onboarding/MyActivity.kt start=start_example end=end_example
```

That's it, if you build and run the app you should see "Hello World!" displayed on the screen.
