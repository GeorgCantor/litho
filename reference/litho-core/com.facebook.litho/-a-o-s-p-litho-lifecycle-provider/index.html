AOSPLithoLifecycleProvider

open class AOSPLithoLifecycleProvider(val lifecycleOwner: LifecycleOwner, delegate: LithoLifecycleProviderDelegate = LithoLifecycleProviderDelegate()) : LithoLifecycleProvider, LifecycleEventObserver, AOSPLifecycleOwnerProvider

This LithoLifecycleProvider implementation dispatches to the registered observers the lifecycle state changes triggered by the provided LifecycleOwner. For example, if a Fragment is passed as param, the observers will be registered to listen to all of the fragment's lifecycle state changes.
