RecyclePool

open class RecyclePool<T : Any>(_name: String, _maxSize: Int, isSync: Boolean) : PoolWithDebugInfo

Used to recycle objects in Litho. Can be configured to be either synchronized or not. A RecyclePool will keep track of its own size so that it can be queried to debug pool sizes.
